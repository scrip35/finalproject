О проекте

Предыстория

Все началось с того, что я случайно наткнулся на анонс от Astana Hub о грантовой программе, направленной на поддержку различных профессий в сфере информационных технологий. Это было словно миг вдохновения – момент, когда возможности раскрыть свой потенциал казались безграничными. Страничка сайта, сверкающая яркими цветами предложений и обещаний, словно манит войти в игру. И я не смог устоять перед этим вызовом. Ведь как часто бывает, возможности для роста и самореализации не стучатся в дверь с такой яркостью и открытостью.

Представьте себе: шанс на обучение в самом центре инноваций, под руководством ведущих экспертов отрасли, обещание не просто получить новые знания, но и применить их на практике в реальных проектах! Мечты о новых навыках, о вдохновляющих коллегах, о возможности внести свой вклад в мир технологий – все это стало частью реальности благодаря этому моменту. И я не мог пройти мимо.

Что мы изучали

HTML

Язык разметки, используемый для создания структуры веб-страниц

CSS

Каскадные таблицы стилей. Используется для стилизации и оформления веб-страниц

Bootstrap

Фреймворк для разработки адаптивных и кроссбраузерных веб-приложений, основанный на HTML, CSS и JavaScript

JavaScript

Язык программирования, применяемый для добавления интерактивности на веб-сайтах.

VueJS

Прогрессивный JavaScript-фреймворк для создания пользовательских интерфейсов и одностраничных приложений

ReactJS

Основная концепция ReactJS - это создание виртуального DOM (Document Object Model), который эффективно обновляется при изменениях данных, что повышает производительность веб-приложений

NodeJs

Среда выполнения JavaScript, позволяющая запускать код на сервере

ExpressJs

Веб-фреймворк для Node.js, упрощающий разработку веб-приложений и API

MongoDB

Документо-ориентированная NoSQL база данных, используемая для хранения данных в JSON-подобном формате
